

# IMDB Dataset

## Data Source

Top 250 as rated by IMDb Users
https://www.imdb.com/chart/top?ref_=ft_250

## Data fields
* `rank` - String. e.g. `1.`
* `name` - String. e.g. `The Shawshank Redemption`
* `year` - String. e.g. `1994`
* `score` - Float. e.g. `9.2`

## Data Volume
250 rows 4 columns
